# Nagpur Zone Sports 2026 - Redesign

This is the official source code for the "Next Gen" Nagpur Zone Sports website.

## Features
- **Cyber/Sport Aesthetic**: Modern Dark Mode with Neon accents.
- **Live Data**: Fetches scores directly from the Google Sheet (CSV).
- **Zero Maintenance**: No backend required. run on GitHub Pages.

## How to Edit
1.  **Code**: All styles are in `css/style.css`.
2.  **Logic**: Application logic is in `js/app.js`.
3.  **Data**: Sheet IDs are configured in `js/data-service.js`.

## How to Deploy (GitHub Desktop)

Since you have **GitHub Desktop** installed, following these steps to publish your site:

1.  **Open GitHub Desktop**.
2.  **Add Local Repository**:
    *   Go to **File** > **Add Local Repository...**
    *   Click **Choose...** and select this folder: `nagpur-sports`.
    *   Click **Add Repository**.
3.  **Create Repository**:
    *   You will see a warning "This directory does not appear to be a Git repository."
    *   Click **create a repository** (blue text link).
    *   Click **Create Repository** in the popup.
4.  **Publish**:
    *   Click the **Publish repository** button in the top bar.
    *   Name it `nagpur-sports` (or any name you prefer).
    *   Uncheck "Keep this code private" (GitHub Pages requires it to be public for the free tier).
    *   Click **Publish Repository**.
5.  **Enable Pages**:
    *   Go to your repository on [GitHub.com](https://github.com).
    *   Go to **Settings** > **Pages**.
    *   Under **Branch**, select `main` and save.
    *   Your site will be live!
