# Nagpur Zone Sports 2026 - Redesign

This is the official source code for the "Next Gen" Nagpur Zone Sports website.

## Features
- **Cyber/Sport Aesthetic**: Modern Dark Mode with Neon accents.
- **Live Data**: Fetches scores directly from the Google Sheet (CSV).
- **Zero Maintenance**: No backend required. run on GitHub Pages.

## How to Edit
1.  **Code**: All styles are in `css/style.css`.
2.  **Logic**: Application logic is in `js/app.js`.
3.  **Data**: Sheet IDs are configured in `js/data-service.js`.

## How to Deploy (GitHub Pages)

Since you don't have `git` installed locally, you can upload manually:

1.  Go to [GitHub.com](https://github.com) and create a **New Repository** named `nagpur-sports`.
2.  Click **"uploading an existing file"** link on the empty repo page.
3.  Drag and drop **ALL** files from this folder (`index.html`, `css/`, `js/`) into the browser window.
4.  Commit the changes.
5.  Go to **Settings** > **Pages**.
6.  Under **Branch**, select `main` and save.
7.  Wait 1 minute, and your site will be live!
